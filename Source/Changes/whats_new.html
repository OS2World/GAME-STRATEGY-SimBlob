What's New with SimBlob
4 September 1998
1: Began implementing the lumber/logging industry. This implementation can be extended to firefighting and building, so I plan to remove the old firefighting and building code and replace it with a single blob scheduler that can handle lumberjacks, firefighters, and builders. The new implementation should be smaller and faster.
3 September 1998
1: Worked out details of the lumber/logging industry.
19 August 1998
1: Improved the terrain generation algorithms. There should no longer be obvious vertical and horizontal lines on the initial map.
2: Changed the firefighter routines to disable themselves when there are no fires around. This speeds up the game by 10 to 15 percent (when there are no fires on the map).
18 August 1998
1: Found and (hopefully) fixed a bug that caused blobs to walk in weird paths when their destination was blocked.
17 August 1998
1: Designed parts of the lumber/logging industry for BlobCity. Forests will be needed to build and sustain a city, so the player will have to conserve some forests to survive.
15 August 1998
1:
(Internal)		 Cleaned up and removed old code that was not being used.
7 August 1998
1:
(Internal)		 Cleaned up a lot of code internally and added more checking to help me find any memory errors.
